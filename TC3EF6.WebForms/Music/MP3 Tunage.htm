MPEG Audio Layer 3 Links...

A typical 4 minute song digitally recorded from an audio CD results in a WAV file 40-50 MB big. MPEG Audio Layer-3 technology reduces this size to 4-5 MB, by compressing the WAV file using the MP3 format. You need a special player, to play these files though...
