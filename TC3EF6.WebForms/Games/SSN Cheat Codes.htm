Tom Clancy SSN was programmed with a set of 'cheat' codes which enable certain unconventional gameplay functions. We will be posting them on this page, six or seven at a time, on a weekly basis. There are 28 in all, so visit this page once a week for a month and you'll have them all.

In order to activate a cheat code, you must follow these steps exactly:

  1. While a mission is running, type "b". This activates the COMM panel.
  2. Type "CISCO SEZ", then hit ENTER. This enables the 'cheat' mode.
  3. Type in the cheat code, then hit ENTER again.
  4. Repeat step 3 to engage other cheat codes, or to repeat a code in the case of a toggle.

WARP TO WAYPOINT

Several customers have contacted us looking for a way to save the game between missions. SSN was designed as a true-to-life primer on submarine command strategies. The missions themselves are structured to impart a certain set of skills to the players over their course, and it was felt at the time that players should be required to play through them in their entirety.

Nonetheless, one of the cheat codes will allow players to skip through 'waypoints,' of which there are about four per mission. Although this isn't exactly the same as saving a mission in progress, it will allow a player to quickly return to a particular spot that they may find especially difficult.

Follow the instructions above and type "WARP TO WAYPOINT" for the cheat code. What this does, essentially, is to mark all of your waypoint objectives as complete, effectively skipping ahead to the end of your current waypoint. Next, type "w". This advances you to the next waypoint.

***NOTE: Repeating this process WILL enable you to skip all the way through to the end of the game. Be forwarned, however, that abusing this cheat will deprive you of the necessary experience that you will need to survive the later missions.***
