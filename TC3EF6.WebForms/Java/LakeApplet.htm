<applet codebase="http://clark.wayne.sss.sungard.com/Java/" code="Lake"
width="392" height="380">
<param name="image" value="dusk.gif">
<param name="overlay" value="SunGard75.gif">
<param name="href" value="http://www.wayne.sss.sungard.com/">
<p>Must use a Java-enabled Browser to display this image. </p>
</applet>

Where:

codebase="http://clark.wayne.sss.sungard.com/Java/"
The directory where the Lake.class file resides. This would be the directory on your own machine if you steal the Lake.class file...
code="lake"
This is the name of the applet
width/height
The dimensions of the GIF image (note the height should be roughly double that of the original GIF file to allow for the rippling effect)
param name="image" value="dusk.gif"
Represents the main GIF image to use in the reflection (transparent GIFs don't look so good)
param name="overlay" value="SunGard75.gif"
Represents the GIF file to be used as an overlay of the reflected background (transparent GIFs do look pretty good in this capacity, although animations are not supported).
Note: If you do not want an overlay, omit this parameter all together.
<param name="href" value="http://www.wayne.sss.sungard.com/">
Represents the hyperlink where the browser will go if the user clicks on the image.
