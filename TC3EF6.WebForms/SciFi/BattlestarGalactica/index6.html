With the rumoured Galactica revival and an increase in fan activity, BG-related software has started to pop up:

The Mac Galactica Pack [NEW!]

A collection of startup screens, sounds, and icons for the Macintosh.

Viper DELUXE

a Battlestar Galactica game for Windows (version 1.4 updated March 12, 1995)

A Battlestar Galactica Theme

(for Windows 95 Plus!)

Another Battlestar Galactica Theme

(for Windows 95 Plus! again)

The Battlestar Font

Macintosh and Windows versions of the typeface used in the opening credits.

The Galactican Font

Macintosh and Windows versions of the typeface used on the Galactica.

The BG Opening Theme Music

A MIDI file of dubious quality. You'll need MIDI software/hardware to play this. (If anyone finds a good BG MIDI file, please send it to me)

GalaktikaMix

This is a weird pop/rap mix of the Galactica theme, in MOD file format. MOD is an Amiga music format, but players are available for Mac and PC.
