A Battlestar Galactica Theme for Windows 95 Plus!

This theme will change your wallpaper, icons, animated cursors and sound scheme to a complete Battlestar Galactica theme. Requires: Windows 95 with Plus!

In order to install the Battlestar Galactica Theme, just copy all the files in the ZIP archive file into your [drive]:\Plus!\Themes directory. Then, in Control Panel, select Desktop Themes and choose the Battlestar Galactica theme from the drop down menu.
