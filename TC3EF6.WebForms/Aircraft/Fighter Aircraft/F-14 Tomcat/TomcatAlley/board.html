This page is intended to give you some informations about Statistics Area, and to help you use it at its maximum capabilities. The Statistics Area is an invaluable tool for keeping track of the hits to the web pages. It will interpret the server's logs and present them to you in a much more useful and friendly form.

In order to get the reports, please type your password in the Pass edit field.
Take into considerations the fact that the hits from company Intranet are ignored by default in all raports.

Help on using Statistics Area

The browser window is divided into two sections: a Control Frame and a Result Frame. The Result Frame is where the answer of your queries will appear. We'll discuss now the controls from the Control Frame:

  • The set of radio buttons for choosing the type of statistic.
    • Raw data: the data will be presented without any interpreting, in the original form, but using colors: black for fulfilled requests, red for errored requests, blue for redirected/check_only request.
    • Top 50 pages: the most visited 50 pages will be presented.
    • Top 50 visitors: the hosts which come here very often.
    • Hits Reports: some statistics regarding the hits.
    • Stats&Averages: general numbers, like total size of transferred data, etc.
  • The Path edit control: you can write there the path on which you want the statistics. If you want statistics on a directory and its subdirectories, write the path to that subdirectory. For a single file, write the path to that file.
  • The Time edit control: you can specify there the time interval on which you want the reports. The correct format is: "dd/Mmm/yyyy" - for the interval begining with that date until today, or "dd/Mmm/yyyy-dd/Mmm/yyyy".
  • The Pass edit control: write there the identification password. If you aren't a registered user, use the "guest" password to get statistics on the path "/~cata" with intranet excluded.
  • The set of check boxes, for the customization of the query.
    • Html files only: the only hits considered for creating the statistic will be on the html files.
    • Today's reports: the time interval will be 1 day, begining and ending with today.
    • One day reports: the time interval will be 1 day, begining with the date from the Time edit field.
  • The Submit and Clear buttons, for submitting, respectively clearing the form.
