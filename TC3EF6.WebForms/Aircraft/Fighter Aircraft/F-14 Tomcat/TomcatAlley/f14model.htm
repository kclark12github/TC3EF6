F-14 Tomcat Kit Details

At present this page features only bare information about all the 1/72nd F-14 model kits that I know of in existence. It will be expanding to cover other scales and to give more detailed reviews of each kit. If anyone has any information/reviews they would like to submit please e-mail me.

Kit information is in the current format:- first the F-14 variant the kit builds into, then the product code of the manufacturer, finally any details I have about the colour scheme/equipment/weapons included with the kit.
