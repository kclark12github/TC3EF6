Aircraft Designations

Prior to September 18, 1962 the U.S. Navy maintained a naming convention based on the following format: <MissionCode><Aircraft Sequence><ManufacturerCode>. The U.S. Army Air Corps had their own naming convention based on a "P" prefix denoting pursuit aircraft, and "B" for bomber aircraft. Any further discussion of nomenclature used by non-Navy branches of the military are beyond this researcher's understanding...

Examples:  
 
Grumman "F4F" Wildcat Fighter
F = Fighter Mission Code
4 = The fourth Grumman fighter aircraft developed for the Navy
F = Manufacturer Code for Grumman
  Vought "F4U" Corsair Fighter
F = Fighter Mission Code
4 = The fourth Vought Fighter developed for the Navy
U = Manufacturer Code for Vought
  Douglas "SBD" Dauntless Scout Bomber
SB = Scout/Bomber Mission Code
No model number code since it was the first Douglas Scout Bomber
D = Manufacturer Code for Douglas

Curtiss "SB2C" Helldiver Scout Bomber

SB = Scout/Bomber Mission Code
2 = The second first Curtiss Scout Bomber developed for the Navy
C = Manufacturer Code for Curtiss
 
  Grumman "TBF" Avenger Torpedo Bomber
TB = Torpedo Bomber Mission Code
No model number code since it was the first Grumman Torpedo Bomber
F = Manufacturer Code for Grumman
 
Note the "TBM" Avenger was the same aircraft, but built by Eastern Motors, and therefore includes an "M" manufacturer code to distinguish these aircraft from the Grumman built models

On September 18, 1962, the unified code with which we are familiar was established. The table below makes reference to these different naming conventions.
